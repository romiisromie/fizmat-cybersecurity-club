# 🛡️ FIZMAT Cybersecurity Club

Welcome to the official repository of the **Cybersecurity Club** at the National School of Physics & Math (FIZMAT) in Almaty, Kazakhstan. 

This repository is an open-source educational hub and laboratory created to help student developers understand computer networks, cryptography, secure coding practices, and Artificial Intelligence (AI) safety.

## 🎯 Our Mission
To build a strong technical community within FIZMAT where students can bridge the gap between competitive mathematics, software engineering, and digital defense. We focus on hands-on labs and real-world problem solving.

## 📅 What We Study (Syllabus Preview)

*   **Introduction to Security:** Understanding how networks work, analyzing traffic using Wireshark, and learning Linux basics.
*   **Web Application Security:** Studying the OWASP Top 10 vulnerabilities (like SQL Injections and XSS) and how to fix them.
*   **AI Safety & Defenses:** Exploring how to protect Large Language Models (LLMs) from prompt injections and secure modern AI systems.
*   **Algorithmic Practices:** Solving competitive security challenges (CTFs) and practicing algorithms mapped to LeetCode patterns.
